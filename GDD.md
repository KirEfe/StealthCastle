> Версия 0.1 | Статус: в разработке | Движок: Unity 6, 2D URP

## Elevator Pitch
Стелс-платформер в стиле метроидвании где главная суперсила героя —
превращаться в любой предмет окружения. Замок полон стражников, но никто
не заподозрит бочку, канделябр или мешок с зерном.

## Ключевая механика — Маскировка

### Как работает
1. Герой зажимает кнопку → задерживает дыхание → запускается **таймер**
2. Система сканирует спрайты рядом с героем
3. Подходящие объекты подсвечиваются **outline-эффектом**
4. Игрок выбирает в какой предмет превратиться
5. Герой принимает форму предмета → враги не реагируют
6. Маскировка спадает когда:
   - Таймер вышел (кончилось дыхание)
   - Герой начал двигаться
   - Герой уже был замечен в момент активации

### Ограничения (делают механику интересной)
- Работает только **до** обнаружения — нельзя спрятаться когда уже видят
- Движение = мгновенный сброс маскировки
- Таймер = давление, нельзя стоять вечно
- Нужен подходящий предмет рядом — пустая комната не даёт укрытия

### Открытые вопросы
- [ ] Как долго длится таймер? (стартовое значение для теста: 5 сек)
- [ ] Можно ли улучшать длину таймера по ходу игры?
- [ ] Враги видят "новый предмет" если его раньше не было? (реализм vs фан)
- [ ] Что если два одинаковых предмета рядом — оба подсвечиваются?
- [ ] Звук при активации/сбросе маскировки?

## Структура мира — Метроидвания

```
ЗАМОК (открытый мир)
│
├── 🏰 Парадный вход        ← стартовая зона, обучение
├── 🍷 Винные погреба        ← первый "настоящий" стелс
├── 🛡️ Казармы стражи        ← враги сложнее, патрули
├── 📚 Библиотека            ← загадки, лор, секреты
├── 👑 Королевские покои     ← высокий уровень охраны
└── ⚔️ Тронный зал           ← финал, "кровавая баня"
```

Зоны открываются по мере получения новых навыков или ключей.
Можно возвращаться в старые зоны с новыми способностями.

## Враги — базовые типы

| Тип | Поведение | Реакция на маскировку |
|-----|-----------|----------------------|
| Стражник | Патруль по маршруту | Не видит предметы |
| Капитан | Широкий конус зрения | Подозревает "новые" предметы |
| Маг | Видит ауру героя | Видит сквозь маскировку |

*(Маг — поздняя игра, контрмера против злоупотребления механикой)*

## Прогрессия

```
Начало          Середина          Финал
────────────────────────────────────────
Только стелс  → Стелс + инструменты → Открытый бой разблокирован
Короткий таймер → Таймер растёт  → Мастер маскировки
Простые стражи → Патрули, собаки → Маги, элита
```
